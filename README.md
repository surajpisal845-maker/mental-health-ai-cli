#  Mental Health AI Assistant (CLI)

##  Description
This project is a Command Line Interface (CLI) based AI tool that analyzes user input text to detect emotions and provides helpful suggestions.

##  Features
- Emotion Detection (Sad, Happy, Stress, Angry, Neutral)
- Suggestion System
- History Tracking using JSON
- CLI-based interaction
- Error handling and help menu

##  Technologies Used
- Python
- JSON
